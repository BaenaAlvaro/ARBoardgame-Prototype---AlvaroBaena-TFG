Este proyecto es el prototipo diseñado y creado por Álvaro Baena Clemente, con el objetivo de demostrar como sería un juego de mesa físico, pero con características de juego Online gracias a la tecnología XR.



Dicho proyecto es un prototipo, con parte de contenido, pulido y UX que no se ha desarrollado debido al alcance del proyecto.





La versión de UNITY necesaria para poder ejecutar el proyecto sin problema es la siguiente: 6000.0.56f1
